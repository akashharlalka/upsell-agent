Agent tone & personality
Applied to every generated response below. Edit anytime — affects future generations only, not what's already been generated. With no funnel stage in this MVP, tone is what carries hesitant/price-sensitive handling.
Anchor tier (recommended default plan)
The plan/SKU featured whenever a response recommends, compares, or CTAs to a specific plan. Not used for broad "what plans exist" listings, which always show every tier. Change this in one place and it updates everywhere it's referenced below and in the prompt bank.
Standing policies
Applies across every row below, not tied to any one utterance. Not tracked as a per-row tag — it depends on live conversation history the real agent has and this table doesn't.
